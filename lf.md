## git命令

* git init 
* git add
* git commit -m ""
* git pull 
* git pull origin master（强制合并）
* git pull origin master --allow-unrelated-histories（强制合并没有关系的仓库）
* git remote rm origin 移除https push方式
* git remote add origin git地址
* git remote -v 查看push方式是否修改成功
* git push origin master
